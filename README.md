# convert-kivy-app-to-APK
Converts your kivy app into an APK to install in android.

1. Fork & clone my repo
2. Enable Github actions in the repo
3. Replace your main.py/main.kv files in the directory
4. Add any extra requirements to the buildozer.spec (check your imports to make sure you have everything)
5. Push the changes to Github
6. If your build succeeded, download the .ZIP file and extract it
7. Inside there, you will have your apk file
8. If your build did not succeed, please post the error on the issues page for further help. (please include the buildoer error)
9. If you have any questions, comments, concerns, etc, please dont hesitate to tell me
